# UI Design

## Ziel

Clash Tool soll übersichtlich, modern und spielnah wirken. Der Nutzer soll sofort sehen, was als Nächstes sinnvoll ist.

## Hauptbereiche

- Dashboard
- Account-Fortschritt
- Upgrade Queue
- Planner
- Simulation
- Screenshot Import
- Einstellungen

## Dashboard

Zeigt:
- Rathaus-Level
- freie Bauarbeiter
- Laborstatus
- nächste empfohlene Upgrades
- Fortschritt in Prozent

## Upgrade Queue

Funktionen:
- Upgrade hinzufügen
- Upgrade entfernen
- Priorität ändern
- Start- und Endzeit anzeigen

## Planner

Zeigt Empfehlungen nach:
- Effizienz
- Offensive
- Defensive
- Ressourcen
- Zeit

## Simulation

Fragen:
- Wo stehe ich in 7 Tagen?
- Wo stehe ich in 30 Tagen?
- Wann bin ich maxed?

## Designstil

- Dunkles Interface
- Clash-inspirierte Karten
- klare Icons
- wenig Text
- schnelle Entscheidungen
